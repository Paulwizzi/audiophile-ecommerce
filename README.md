Plain JavaScript Next.js 14 project scaffold for the Audiophile home page (landing + navbar + footer + placeholders).

## Quick start

1. Extract the zip (or open this folder)
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the dev server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000

This project uses Tailwind CSS. After installing dependencies, the dev server will pick up Tailwind styles.

Files included:
- pages/index.js (responsive landing page)
- pages/_app.js (global styles)
- components/Navbar.js, Footer.js
- placeholder pages: pages/headphones.js, pages/speakers.js, pages/earphones.js
- public/images: small SVG placeholder assets
